# prr-mode.el

Major mode for editing [.prr](https://github.com/danobi/prr) files in Emacs.

## Installation

**Emacs:**

``` emacs-lisp
(use-package prr-mode
  :straight (:host github :repo "elliotekj/prr-mode.el"))
```

**Doom Emacs:**

``` emacs-lisp
(package! prr-mode
  :recipe (:host github :repo "elliotekj/prr-mode.el"))
```
